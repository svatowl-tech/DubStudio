# 📖 Руководство пользователя Anime Dub Manager (ADM)

Добро пожаловать в **Anime Dub Manager**! Это приложение создано для того, чтобы объединить все инструменты студии озвучки в одном «едином окне». Ниже приведена пошаговая инструкция по работе с программой.

---

## 1. Подготовка (Первый запуск)
Чтобы приложение работало корректно, первым делом зайдите в раздел **«Настройки»** (иконка шестеренки):

*   **Рабочая директория:** Выберите папку, где будут храниться все ваши проекты и файлы.
*   **FFmpeg:** Если у вас есть своя сборка FFmpeg, укажите путь к ней (иначе будет использоваться встроенная).
*   **API Ключ OpenRouter:** Вставьте ваш ключ (формата `sk-or-v1-...`), чтобы активировать умный AI-перевод (Gemini 2.0). Если ключа нет, будет работать обычный Google Translate.
*   **NVENC:** Включите аппаратное ускорение, если у вас видеокарта NVIDIA, чтобы сборка релизов «летала».
*   **База участников:** Импортируйте файл `participants.json` с вашими ребятами. База легко пополняется и редактируется прямо в интерфейсе.

---

## 2. Создание проекта
На Главной панели (**Dashboard**):

1.  Нажмите **«Новый проект»**.
2.  **Поиск:** Вы можете найти аниме в базе **Шикимори** (введите название и нажмите «Найти») — приложение само подтянет постер, синопсис и количество серий.
3.  **Команда:** Назначьте даберов и звукорежиссера на проект по кнопке наверху.
4.  **Персонажи:** Во вкладке «Персонажи» распределите роли по даберам через выпадающий список.
5.  **Серии:** Внутри проекта нажмите «Добавить серию». Загрузите **RAW** (видео без субтитров или хардсаб) и файл субтитров **.ASS**.

---

## 3. Работа с субтитрами (Утилиты ASS)
Перейдите во вкладку **«Утилиты (ASS)»**:

*   **Перевод:** Можно сделать перевод, если есть только японские или английские субтитры.
*   **Разделение:** 
    *   Через выпадающий список можно назначить роли всем персонажам из субтитров. Можно назначать по несколько даберов на одну реплику.
    *   **Разметка реплик:** Если попадаются ошибки в именах или нужно переназначить реплики (если это не сделано в исходных сабах), используйте этот инструмент.
*   **Разделить по даберам:** Нажмите эту кнопку, и приложение создаст отдельные файлы субтитров для каждого участника.
*   **Экспорт Даберам:** Автоматически сконвертирует хардсаб небольшого размера со всеми нужными файлами сабов и приветственным сообщением для начала работы.

---

## 4. Озвучка и Контроль (QA)
*   **Загрузка записей:** Когда даберы присылают файлы, прикрепляйте их к серии во вкладке **«QA Панель»**.
*   **Прослушивание:** Слушайте записи прямо поверх видео. 
*   **Правки (Фиксы):** Если нашли ошибку — начните писать правку и жмите «Отправить». Фикс запишется соответствующему даберу с нужным таймингом.
*   **Выписка фиксов:** Нажмите кнопку, чтобы получить готовое сообщение со всеми правками — останется только скопировать и отправить даберу.
*   **Экспорт для звукача:** После загрузки всех фиксов нажмите кнопку экспорта. Приложение сформирует видео с вшитыми надписями и все удобно поименованные аудиофайлы.

---

## 5. Релиз и Публикация
Когда звукорежиссер закончил работу, перейдите в раздел **«Сборка релиза»**:

*   **Сборка:** Выберите финальную аудиодорожку и нажмите «Собрать релиз». Приложение склеит видео и звук.
*   **Скриншоты:** Можно сразу сделать скриншот серии для обложки релиза.
*   **Посты:** Выберите шаблон (**TG** или **VK**), и система создаст готовый текст поста со списком всей команды.
*   **Быстрый доступ:** Используйте блок «Быстрые ссылки» для мгновенного перехода в кабинеты VK Video, Kodik или Anime365.
