## 1. Начало работы и Настройка
При первом запуске программы необходимо настроить рабочую среду.

### Экран настроек (Settings)
*Место для скриншота: Окно настроек с полями путей*

**Основные параметры:**
*   **Base Directory**: Папка на вашем компьютере, где будут храниться все файлы. Программа автоматически создает там структуру папок для каждого проекта.
*   **FFmpeg / FFprobe Path**: Пути к системным файлам для обработки видео. Обычно определяются автоматически.
*   **GPU Acceleration (NVENC)**: Если у вас видеокарта NVIDIA, включите этот пункт. Это ускорит "вшивание" субтитров (Hardsub) в несколько раз.

**Пошаговый гайд:**
1. Нажмите на иконку шестеренки в верхнем правом углу.
2. Выберите папку для проектов (например, `D:/AnimeProjects`).
3. Нажмите **Save Settings**.

## 2. Управление проектами (Dashboard)
Здесь вы создаете новые тайтлы и добавляете серии.

### Главный экран
*Место для скриншота: Список карточек проектов*

**Функции кнопок:**
*   **Add Project**: Открывает форму создания нового аниме.
*   **Add Episode**: Добавляет новую серию в выбранный проект.
*   **Select RAW**: Выбор исходного видеофайла. Если файл в формате MKV, программа предложит перекодировать его в MP4 для совместимости.
*   **Select Subtitles**: Выбор файла субтитров (.ass). Программа автоматически скопирует его в папку серии.
*   **Иконка "Папка"**: Открывает папку текущей серии в проводнике Windows.

**Пошаговый гайд: Создание серии**
1. Нажмите **Add Project**, введите название аниме.
2. Внутри проекта нажмите **Add Episode**.
3. Нажмите **Select RAW** и выберите видео.
4. Нажмите **Select Subtitles** и выберите скрипт.
5. Теперь серия готова к работе!

## 3. Редактор ролей (Editor)
Этот экран предназначен для распределения персонажей между даберами.

### Интерфейс редактора
*Место для скриншота: Таблица со строками субтитров и колонкой "Actor"*

**Функции кнопок:**
*   **Split by Dubber**: Создает отдельные .ass файлы для каждого дабера, который назначен на роли в этой серии. Файлы сохраняются в папку `dubbers`.
*   **Export Full ASS**: Создает копию основного скрипта, где вместо имен персонажей подставлены ники даберов. Это необходимо звукорежиссеру для сведения.
*   **Save Changes**: Сохраняет ваши правки в тексте или распределении ролей.

**Пошаговый гайд: Подготовка файлов для даберов**
1. Перейдите во вкладку **Editor**.
2. В колонке справа (Roles) назначьте даберов на персонажей.
3. Нажмите **Split by Dubber**.
4. Отправьте полученные файлы из папки `dubbers` своим актерам.

## 4. Инструмент разделения (Splitter)
Если вам не нужно сложное управление проектом, а нужно просто быстро нарезать скрипт по именам персонажей.

### Экран Splitter
*Место для скриншота: Окно Splitter с чекбоксами*

**Настройки:**
*   **Distribute Groups**: Если включено, реплики с именами "Все", "Толпа" попадут во все файлы.
*   **Distribute Multiple Roles**: Если в строке несколько имен (Наруто, Саске), строка попадет обоим.
*   **Output Format**: Выбор между .ass и .srt.

**Пошаговый гайд:**
1. Выберите серию.
2. Настройте чекбоксы.
3. Нажмите **Split by Actor**. Готово!

## 5. Контроль качества и Рендер (QA Panel)
Здесь вы проверяете работу даберов и собираете финальное видео.

### Панель QA
*Место для скриншота: Видеоплеер с аудиодорожками даберов*

**Функции:**
*   **Список дорожек**: Вы видите всех даберов серии. Можно прослушать их записи прямо поверх видео.
*   **Upload Fix**: Если в записи ошибка, вы можете загрузить исправленный файл (фикс) прямо здесь.
*   **Bake Video (Hardsub)**: Запускает процесс "вшивания" субтитров в видео. На выходе вы получаете готовый MP4 файл для публикации.

**Пошаговый гайд: Финальный рендер**
1. Убедитесь, что все субтитры проверены.
2. Перейдите в **QA Panel**.
3. Нажмите **Bake Video**.
4. Дождитесь окончания прогресс-бара. Файл появится в папке серии.

## 6. Команда (Participants)
Ваша база данных контактов.

### Список участников
*Место для скриншота: Таблица с никами и ролями*

*   **Add Participant**: Добавление нового участника (Ник, Telegram, Роль).
*   **Roles**: Укажите, является ли человек дабером, технарем или переводчиком. Это поможет фильтровать список при назначении ролей в проекте.

## 7. Загрузчик (Downloader)
Инструмент для скачивания исходников.

*   Вставьте прямую ссылку на файл (видео или субтитры).
*   Программа скачает его и автоматически положит в нужную папку текущей серии.

## 8. Часто задаваемые вопросы (FAQ)

**В: Почему видео не воспроизводится?**
О: Убедитесь, что файл имеет расширение .mp4. Если у вас MKV, используйте функцию транскодирования в Dashboard.

**В: Где лежат мои файлы?**
О: Путь указан в настройках (Base Directory). Внутри этой папки создаются подпапки для каждого аниме.

**В: Программа зависла при рендере (Bake Video).**
О: Рендер — тяжелый процесс. Если у вас слабое железо, это может занять время. Проверьте, не закончилось ли место на диске.
